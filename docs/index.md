# SmartHive

SmartHive es un ecosistema de agentes inteligentes diseñado para automatizar y optimizar tareas de desarrollo de software.

## Características Principales

### 🤖 Agentes Inteligentes
- **Backend**: Desarrollo y mantenimiento de APIs
- **Frontend**: Desarrollo de interfaces de usuario
- **Database**: Gestión y optimización de bases de datos
- **DevOps**: Automatización de CI/CD y despliegue

### 🔄 Orquestación
- Control centralizado de agentes
- Gestión de recursos y estados
- Comunicación entre agentes

### 🧪 Testing
- [Testing de LLM](testing/llm_testing.md): Evaluación de respuestas de agentes
- Tests unitarios y de integración
- Métricas de calidad

## Inicio Rápido

```bash
# Clonar el repositorio
git clone https://github.com/reyer3/smart_hive.git

# Instalar dependencias
poetry install

# Configurar variables de entorno
cp .env.example .env

# Ejecutar tests
poetry run pytest
```
